# ViLedger 📒

A digital Khata/Udhar app for small shopkeepers to manage customer credit transactions.

## Problem
Small shopkeepers give products on credit and write transactions in notebooks - leading to forgotten records, disputes and lost money.

## Solution
ViLedger is a simple, fast digital credit ledger that helps shopkeepers track credit, payments and balances for each customer.

## Features
- 📱 Phone number login
- 📊 Dashboard with daily summary
- 👥 Customer management
- 💳 Credit transaction recording
- 💰 Payment recording (Cash, UPI, Bank Transfer)
- 📜 Full transaction history per customer

## Tech Stack
- React Native
- Expo
- Expo Router
